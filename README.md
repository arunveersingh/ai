# AI Tooling & Prompts

A curated collection of AI interaction patterns, system prompts, and tooling built from hands-on experience working with large language models in real workflows.

## What this is

Not a tutorial repo. This is a working reference — prompts and tools I've refined through actual use, across multiple AI platforms (Claude, GPT, Grok, Kiro, and others). Each piece exists because it solved a real problem or meaningfully improved how I think with these systems.

## Browse the collection

**[→ Prompt Catalogue](./prompts/CATALOGUE.md)** — full list with descriptions, use cases, and usage instructions.

## Philosophy

Most people use AI as a search engine with better grammar. The leverage comes from designing *how* the model thinks with you — shaping its behavior, calibrating its depth, and building interaction patterns that compound over time.

These artifacts reflect that approach: deliberate, opinionated, and tested against real work.

## Usage

1. Open the [Prompt Catalogue](./prompts/CATALOGUE.md)
2. Pick a prompt that fits your situation
3. Copy the full content into your AI tool as a system prompt
4. Start interacting

Each prompt is portable markdown. Works in Claude, ChatGPT, Grok, Kiro — no vendor lock-in.
